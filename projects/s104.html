Conception et création d'une base de données

A partir de l'histoire du Titanic, nous devions créer une base de données associée. L'objectif était de collecter des informations nous-mêmes, de réfléchir à une bonne organisation pour la base de données avec des tables pertinentes, et de voir si elle était utilisable. Nous étions en binôme et avions plusieurs semaines pour réaliser cette SAE. Elle était découpée en 3 parties avec des résultats intermédiaires à fournir.

Compétences

  • Créer une base de données sous PostgreSQL
  • Schématiser, concevoir l'organisation d'une base de données
  • Trier des données, les représenter sous une autre forme
  • Sélectionner les informations pertinentes (pour concevoir ou interroger)
